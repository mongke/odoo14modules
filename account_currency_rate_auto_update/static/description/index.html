POS Fast Load

This module speeds up POS load time by up to 100 times

Configuration

To configure this module, you need to:

  • Go to Settings > Technical > Scheduled Actions
  • Activate the scheduled action "Refresh POS Cache for Faster Loading"
  • Configure its "Interval Unit" and "Next Execution Date"
  • It is advised to set "Interval Unit" as Days and "Next Execution Date" during non-working hours
